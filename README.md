## ðŸš€ How to Run a Brick

Since this is a monorepo approach for learning, navigate to the specific brick folder:

1.  **Clone the repo:**
    ```bash
    git clone https://github.com/IsaiahDoesDev/Bricks.git
    ```

2.  **Navigate to the brick:**
    ```bash
    cd 01-neon-light
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

5.  Open [http://localhost:3000](http://localhost:3000) with your browser.
